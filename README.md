# Project Sentinel

An AI-powered Swing Trading Intelligence System built with Python.

The goal of Project Sentinel is to help traders identify high-quality swing trading opportunities by combining market analysis, trend detection, risk management, and historical backtesting into a single decision-support tool.

---

## Current Features

✅ US Market Health Analysis (SPY & QQQ)

✅ EMA Trend Analysis
- EMA20
- EMA50
- EMA200

✅ Volume Analysis

✅ ATR Volatility Stop

✅ Swing Support Stop

✅ Reward-to-Risk Calculation

✅ Position Size Calculator

✅ Historical Backtesting

---

## Project Goals

- Scan US stocks automatically
- Rank the best swing trading opportunities
- Explain why a trade is recommended
- Calculate risk automatically
- Perform historical backtesting
- Build an AI-assisted trading dashboard
